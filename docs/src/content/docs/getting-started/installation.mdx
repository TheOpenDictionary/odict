## CLI

Choose the installer for your platform or install from source if you want to build the CLI locally.

<Tabs>
<TabItem label="macOS / Linux">

```bash
curl --proto '=https' --tlsv1.2 -LsSf https://odict.org/install.sh | sh
```

This shell installer downloads the latest release binary for macOS or Linux.

</TabItem>

<TabItem label="Homebrew">

```bash
brew install TheOpenDictionary/odict/odict
```

Homebrew is the simplest option on macOS.

</TabItem>

<TabItem label="Windows">

```powershell
powershell -ExecutionPolicy ByPass -c "irm https://github.com/TheOpenDictionary/odict/releases/latest/download/odict-installer.ps1 | iex"
```

</TabItem>

<TabItem label="From source">

```bash
git clone https://github.com/TheOpenDictionary/odict.git
cd odict
# Requires Rust 1.75+.
cargo install --path cli
```

</TabItem>
</Tabs>

### Verify the CLI

```bash
odict --version
```

## Language bindings

Install the package for the language you are using.

<Tabs>
<TabItem label="Python">

```bash
pip install theopendictionary
```

Requires Python 3.8.1+. See the [Python API docs](/api/python/) for usage.

</TabItem>
<TabItem label="JavaScript">

```bash
npm install @odict/node
```

Requires Node.js 12+. The package includes native binaries for all major platforms. See the [JavaScript API docs](/api/javascript/) for usage.

</TabItem>
<TabItem label="Rust">

Add the crate to your `Cargo.toml`:

```toml
[dependencies]
odict = "3"
```

See the [Rust API docs](/api/rust/) for usage and feature flags.

</TabItem>
</Tabs>
